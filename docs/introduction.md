# Introduction

Hallucinator is a tool that validates academic citations against multiple databases to detect fabricated references, retracted papers, and author mismatches.

See the project [README](https://github.com/gianlucasb/hallucinator), [MANIFESTO](https://github.com/gianlucasb/hallucinator/blob/main/MANIFESTO.md), and [Rust README](https://github.com/gianlucasb/hallucinator/blob/main/hallucinator-rs/README.md) on GitHub for project context.
